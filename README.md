# Проектная работа: «Закрывающий тег»

Интерактивный адаптивный сайт-галерея, представляющий собой концептуальный блог с пиксель-арт иллюстрациями. Проект демонстрирует продвинутые техники современной верстки, работу с анимациями и кастомными UI-элементами.

## Технологии и особенности проекта

* **Адаптивная резиновая верстка**: интерфейс плавно масштабируется от мобильных экранов (375px) до десктопа с помощью современной CSS-функции `clamp()`.
* **Вариативные шрифты**: интеграция шрифта *Inter* с динамической настройкой насыщенности текста через свойство `font-variation-settings`.
* **Сложные CSS-эффекты и графика**:
  * Кастомный фиксированный фон, имитирующий сетку ЭЛТ-монитора, созданный с помощью `repeating-linear-gradient`.
  * Режимы смешивания цветов `mix-blend-mode: difference` и `hard-light` для инверсии элементов и стилизации водяных знаков.
  * Продвинутая обводка текста с использованием `@supports` и безопасных фоллбэков (`text-shadow`).
* **Оптимизация SVG-графики**: сборка векторных иконок в производительные `<symbol>`-спрайты для повторного использования.
* **Интерактивность**:
  * Модальное всплывающее окно на базе семантического HTML-тега `<dialog>`.
  * Плавные анимации появления элементов (`transition`) и покадровая анимация пульсации лайка (`@keyframes`).
  * JS-скрипты для симуляции лайков карточек и управления диалоговым окном.

## 🚀 Как запустить проект локально

1. Склонируйте репозиторий на свой компьютер:
   ```bash
   git clone https://github.com/ZakharovVlr/zakrivayuschiy-teg-f.git
   ```
2. Перейдите в папку с проектом:
   ```bash
   cd zakrivayuschiy-teg-f
   ```
3. Откройте файл `index.html` в любом современном браузере (рекомендуется использовать расширение *Live Server* в VS Code).
